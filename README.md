# Sistema de Oficina Mecânica - Modelo Conceitual

Este projeto apresenta o modelo conceitual de um sistema de controle e gerenciamento de ordens de serviço para uma oficina mecânica.

## Contexto

Clientes levam seus veículos à oficina para manutenção ou revisão. Cada veículo é analisado por uma equipe de mecânicos que identifica os serviços necessários e gera uma ordem de serviço (OS).

A OS contém:

* Serviços a serem realizados
* Peças utilizadas
* Valores de mão de obra
* Datas de emissão, entrega e conclusão
* Status da ordem

O valor total da OS é calculado com base nos serviços executados e nas peças utilizadas.

## Principais Entidades

* Cliente
* Veículo
* Ordem de Serviço
* Mecânico
* Equipe
* Serviço
* Peça

## Objetivo

Permitir o controle completo das ordens de serviço, equipes responsáveis, serviços executados e peças utilizadas no conserto ou manutenção de veículos.
