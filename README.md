# Forma-o-SQL-Database-Specialist


Projeto de Banco de Dados – E-commerce
📌 Descrição do Projeto

Este projeto consiste na modelagem conceitual e lógica de um banco de dados para um sistema de E-commerce. O objetivo é representar as principais entidades e relacionamentos envolvidos em uma plataforma de vendas online, permitindo o gerenciamento de clientes, pedidos, produtos, pagamentos, entregas e estoque.

O modelo foi desenvolvido como parte de um desafio acadêmico, aplicando conceitos de modelagem de dados, normalização e relacionamento entre entidades.

No repositório há mais dois arquivos relacionados a construção do desafio. "Ordem de Serviço" e "Universidade Refinado".
