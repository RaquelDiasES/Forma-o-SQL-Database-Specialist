# Modelo de Banco de Dados – E-commerce

## Descrição

Este projeto apresenta um modelo de banco de dados relacional para um sistema de **E-commerce**.
O objetivo do modelo é estruturar e organizar as principais informações necessárias para o funcionamento de uma plataforma de vendas online, incluindo clientes, produtos, pedidos, pagamentos, entregas, fornecedores e controle de estoque.

O modelo foi desenvolvido utilizando um **Diagrama Entidade-Relacionamento (DER)**, representando as relações entre as diferentes entidades do sistema.

## Entidades do Modelo

O banco de dados é composto pelas seguintes entidades:

* Cliente
* Cliente PF
* Cliente PJ
* Pedido
* Item Pedido
* Pagamento
* Entrega
* Produto
* Categoria
* Estoque
* Estoque Produto
* Fornecedor
* Disponibilizando um produto
* Terceiro - Vendedor
* Produtos por Vendedor (terceiro)

## Objetivo do Modelo

Este modelo permite gerenciar:

* Cadastro de clientes (pessoa física e jurídica)
* Controle de produtos e categorias
* Gerenciamento de pedidos e seus itens
* Registro de pagamentos
* Controle de entregas
* Controle de estoque
* Relacionamento com fornecedores
* Venda de produtos por vendedores terceiros
